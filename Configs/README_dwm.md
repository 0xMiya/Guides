# My dwm build

## Patches

I have installed several patches:

* dwm-status2d
* dwm-push\_no\_master
* dwm-attachside
* dwm-centeredmaster
* dwm-deck
* dwm-deck-double
* dwm-pertag
* dwm-fullgaps
* dwm-alternativetags
* dwm-6.2-urg-border

All these patches can be found on the official suckless website, under dwm->patches

You can install a patch using:

	patch -p1 < <patch.diff>
